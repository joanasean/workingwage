Welcome to Project 2! In this project we are visualizing the average hourly wage of each profession collected by the Bureau of Labor and Statistics.
Below we have the following visuals created using D3. The data was cleaned using pandas and exported into a json format. Please click on the image of
each one below to see the visual. The sunburst allows for a detailed disaggregation of the data, as compared to the heatmap, which gives a high level
view of each state.
For our external library we used Masonry JS which allows for a clean grid layout of images that adjusts to the size of the photo library.
